## Import Instructions

To import these games into the database:

1. Ensure you have a GitHub token set: `export GITHUB_TOKEN=your_token`
2. Run the fetch script: `npm run fetch`
3. Parse the output: `npx tsx scripts/parse-games-list.ts data/fetched-games.md`
4. Sync to database: `npm run sync:local`

Or manually add the games above to a markdown file and use the parse-games-list.ts script.
