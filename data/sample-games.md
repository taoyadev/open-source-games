# Open Source Games

A curated list of open source games.

## Action games

- **[Doom](https://www.doom.com)** - The classic first-person shooter. [[source]](https://github.com/id-Software/DOOM)
- **[ Abuse ](http://abuse.zoy.org)** - A side-scrolling platform shooter. [[source]](https://github.com/anarkiaviene/abuse)
- **Alien Arena** - A fast-paced first-person shooter. [[source]](https://github.com/code-disaster/aa-source)
- **[OpenArena](https://openarena.ws)** - A community-produced deathmatch FPS. [[source]](https://github.com/OpenArena/engine)
- **AssaultCube** - A tactical first-person shooter. [[source]](https://github.com/assaultcube/AC)

## Adventure games

- **[Endless Sky](https://endless-sky.github.io/)** - Space exploration, trading, and combat game. [[source]](https://github.com/endless-sky/endless-sky)
- **ScummVM** - A program which allows you to run certain classic graphical point-and-click adventure games. [[source]](https://github.com/scummvm/scummvm)
- **Pioneer** - A game of lonely space adventure. [[source]](https://github.com/pioneerspacesim/pioneer)

## Business and Tycoon games

- **[OpenTTD](https://www.openttd.org)** - An open source simulation game based upon Transport Tycoon Deluxe. [[source]](https://github.com/OpenTTD/OpenTTD)
- **[Factorio](https://factorio.com)** - A game about building and maintaining factories. [[source]](https://github.com/factorio/factorio)

## City-Building games

- **[OpenCity](http://www.opencity.info/)** - A 3D city simulator game. [[source]](https://github.com/sohilab/open-city)

## First-Person games

- **[Red Eclipse](https://www.redeclipse.net)** - A fun-filled new take on the first person arena shooter. [[source]](https://github.com/redeclipse/base)

## Platformers

- **[Super Mario War](http://www.smwstuff.net)** - A multiplayer game where the goal is to jump on your enemies. [[source]](https://github.com/mmatyas/supermariowar)
- **[The Battle for Wesnoth](https://www.wesnoth.org)** - A turn-based tactical strategy game. [[source]](https://github.com/wesnoth/wesnoth)

## Puzzle games

- **[Enigma](http://www.nongnu.org/enigma/)** - A puzzle game reminiscent of Oxyd and Rock'n'Roll. [[source]](https://github.com/Enigma-Game/Enigma)
- **[Neverball](http://neverball.org)** - A tilt-based puzzle game. [[source]](https://github.com/Neverball/neverball)

## Racing games

- **[Trigger Rally](http://trigger-rally.sourceforge.net)** - A free rally simulation game. [[source]](https://github.com/trigger-rally/trigger-rally)

## Real-Time strategies

- **[0 A.D.](https://play0ad.com)** - A free, open-source game of ancient warfare. [[source]](https://github.com/0ad/0ad)
- **[Warzone 2100](http://wz2100.net)** - A cross-platform real-time strategy game. [[source]](https://github.com/Warzone2100/warzone2100)

## Roguelikes

- **[Dungeon Crawl Stone Soup](https://crawl.develz.org)** - A classic roguelike game. [[source]](https://github.com/crawl/crawl)
- **[NetHack](http://www.nethack.org)** - A classic roguelike dungeon crawler. [[source]](https://github.com/NetHack/NetHack)

## Role-Playing games

- **[Flare RPG](http://flarerpg.org)** - Free Libre Action Roleplaying Engine. [[source]](https://github.com/flareteam/flare-game)
- **[Veloren](https://veloren.net)** - An open-source voxel RPG. [[source]](https://github.com/veloren/veloren)

## Sandbox games

- **[Minetest](https://www.minetest.net)** - An open-source voxel game engine. [[source]](https://github.com/minetest/minetest)
- **[Terasology](http://terasology.org)** - A voxel world game. [[source]](https://github.com/MovingBlocks/Terasology)

## Shoot 'em up games

- **[R-Type clone](https://github.com/leirrel/R-Type-Clone)** - An R-Type clone. [[source]](https://github.com/leirrel/R-Type-Clone)

## Sport games

- **[Slipstream 5000](http://www.slipstream5000.com)** - A racing game. [[source]](https://github.com/slipstream5000/sl5000)

## Third-Person games

- **[VDrift](http://www.vdrift.net)** - A racing simulation. [[source]](https://github.com/VDrift/vdrift)

## Turn-Based strategies

- **[FreeCol](http://www.freecol.org)** - A turn-based strategy game based on the old game Colonization. [[source]](https://github.com/FreeCol/freecol)
- **[FreeCiv](http://www.freeciv.org)** - A free turn-based multiplayer strategy game. [[source]](https://github.com/freeciv/freeciv)
